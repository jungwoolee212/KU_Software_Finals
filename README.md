# 📸 사진 기반 음악 추천 앱 (Photo to Music Recommendation App)

사용자가 업로드한 사진의 분위기와 상황을 AI가 분석하여, 그 사진과 가장 잘 어울리는 **음악 3곡을 추천**해주는 웹 애플리케이션입니다.

## 🚀 프로젝트 개요
* **목적:** 사진과 어울리는 감성적인 음악 추천 서비스 제공
* **주요 기능:**
  * 이미지 업로드 (JPG, PNG 등) 및 미리보기
  * Google Gemini AI를 활용한 이미지 분석
  * 사진 분위기에 맞는 노래 3곡 및 추천 이유 제공

## 🛠 사용 기술 스택
* **언어:** Python 3.x
* **웹 프레임워크:** Streamlit (빠르고 직관적인 UI 구성)
* **인공지능 API:** Google Gemini API (`gemini-1.5-flash` 모델)
* **이미지 처리:** Pillow (PIL)

## ⚙️ 설치 및 실행 방법

> ⚠️ **[중요 보안 안내] API 키 미포함 사유**
> 본 프로젝트는 평가 및 GitHub 공개용으로 작성되었으므로, **보안상의 이유로 소스코드 내에 API 키를 하드코딩하여 포함하지 않았습니다.** 
> API 키가 퍼블릭 저장소에 유출될 경우 심각한 보안 사고나 과금이 발생할 수 있기 때문입니다. 
> 앱을 실행하여 평가하실 때는, 번거로우시더라도 아래 안내에 따라 무료 API 키를 발급받아 앱 실행 화면에서 직접 입력해 주시기 바랍니다.

### 1. 사전 준비 (API 키 발급)
이 앱을 실행하려면 Google의 **무료** Gemini API 키가 필요합니다.
1. [Google AI Studio](https://aistudio.google.com/)에 접속하여 구글 계정으로 로그인합니다.
2. `Get API key` 메뉴에서 새로운 API 키를 생성하고 복사해둡니다.

### 2. 패키지 설치
터미널(또는 명령 프롬프트)을 열고, 이 프로젝트 폴더로 이동한 뒤 아래 명령어를 입력하여 필요한 라이브러리를 설치합니다.
```bash
pip install -r requirements.txt
```

### 3. 애플리케이션 실행
설치가 완료되면 아래 명령어로 앱을 실행합니다.
```bash
streamlit run app.py
```
명령어를 실행하면 자동으로 웹 브라우저가 열리며 앱이 실행됩니다. (기본 주소: `http://localhost:8501`)

### 4. 사용 방법
1. 웹 브라우저 좌측 사이드바에 복사해둔 **Gemini API 키**를 입력합니다.
2. 화면 중앙의 `Browse files` 버튼을 눌러 원하는 사진을 업로드합니다.
3. **"이 사진에 어울리는 음악 추천받기"** 버튼을 클릭합니다.
4. AI가 분석한 추천 음악 3곡과 그 이유를 확인합니다!

## 💡 주요 특징 및 기대 효과
* **코드에 하드코딩 없는 보안 구조:** 사용자가 화면에서 직접 API 키를 입력하도록 설계하여, GitHub에 코드를 공개해도 보안 문제가 발생하지 않습니다.
* **직관적인 UI:** Streamlit을 활용하여 누구나 쉽게 사용할 수 있는 깔끔한 인터페이스를 구현했습니다.
* **빠른 멀티모달 AI:** 최신 `gemini-1.5-flash` 모델을 사용하여 이미지를 빠르게 분석하고 자연스러운 한국어로 결과를 제공합니다.
